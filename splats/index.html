Hello World with Gaussian Splats

Controls: On desktop, use WASD to move, mouse drag to look around, and Q/E to move up/down. On mobile, use touch drag to look around and pinch to zoom.

This is showing a basic point cloud extracted from a 3D Gaussian Splat. The splat was generated from a single photo using Apple's awesome ml-sharp library.

Try it yourself: Clone the ml-sharp repo, follow the README (or ask Claude to help you follow it), then use it to convert your own image to a Gaussian splat. Once you have a .ply file, ask Claude to generate a simple web viewer that renders the splat as a point cloud with interactive controls.

Note that rendering these splats properly with a tool like SuperSplat will produce a much better result!
