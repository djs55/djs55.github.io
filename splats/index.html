Hello World with Gaussian Splats

Controls: Use WASD to move around, Q/E to move up/down, drag to look around, and scroll to zoom. On mobile, touch drag to rotate and pinch to zoom.

This is showing a 3D Gaussian Splat rendered using GaussianSplats3D. The splat was generated from a single photo using Apple's awesome ml-sharp library.

Try it yourself: Clone the ml-sharp repo, follow the README (or ask Claude to help you follow it), then use it to convert your own image to a Gaussian splat. Once you have a .ply file, you can use viewers like SuperSplat or GaussianSplats3D to display it.
